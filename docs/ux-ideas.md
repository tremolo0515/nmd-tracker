# UX改善アイデア集

## 通常UX案（実装待ち）

1. **セルタップ時のリップルエフェクト**
タップした瞬間にそのセルのアクセントカラーで波紋が広がる。「記録できた」という触覚的フィードバックの視覚補完。

2. **確率が上がる瞬間のカード発光**
ハズレを記録してNが増加し確率が上がった時、そのポートレートカードが一瞬強く光る。「次はチャンス」という示唆。

3. **100%確定時の演出**
N=2になった瞬間に「確定」テキストがパルスし、カードの枠がゆっくり点滅する。次の記録へのモチベーション。

4. **月スワイプ時のページめくり感**
現在は opacity の変化だけ。隣月のカードがわずかに奥に引っ込むような perspective 変形を加えると、空間的な位置感が出る。

5. **「今月」に戻るボタン**
過去月を見ていると現在月に戻れない。ヘッダーの月ラベルをタップ、またはダブルタップで今月にジャンプ。

6. **記録の振り返りサマリー**
下部に小さく「直近Nヶ月: 出現X回 / ハズレY回」を表示。長く使うほど蓄積データに意味が出る。

8. **記録ミス時の取り消し**
誤タップで「出現」→「ハズレ」に変えてしまった場合のアンドゥ。シェイクジェスチャーかヘッダーの戻るアイコンで直前の操作を1つ戻す。

9. **ニュームーンデー開催中バナー**
イベント期間中に「開催中」バッジをヘッダーに表示。アプリを開いた時に「今日記録できる日か」が即座にわかる。（開催日データの管理が必要）

10. **確率に応じたポートレートの輝き強度**
16.7%は暗め、50%は通常、100%は強い内側グローを加える。データを見なくても視覚的に「今月チャンス」とわかる。

## レア演出案

1. **赤い流れ星**
通常の白い星が、ごく低確率（1〜2%）で赤くなり、画面を横切るアニメーションが走る。ポケモンスリープの「赤い月」イベントのオマージュ。

2. **流星群**
数百回に1回のアプリ起動時、星が一斉に同じ方向へ流れる「流星群モード」が数秒発生する。気づかないユーザーも多い密度で仕込む。

3. **ポートレートが瞬きする**
ミュウやダークライの画像が、ごく稀（数分に1回）に一瞬だけ目を閉じたような暗転をする。画像の差し替えではなく、overlay の一瞬フラッシュで表現可能。

4. **セルが震える**
100%確定の翌日（N=0リセット後の最初のタップ）の瞬間だけ、セルが小さく震えてから記録される。「また一から始まる」という感覚の演出。

5. **夜明け演出**
ものすごく久しぶり（例: 30日以上）にアプリを開いた時、背景が一瞬明るくなってから夜空に戻る。「帰ってきた感」の演出。

6. **隠しメッセージ**
ある特定の操作パターン（例: 3ポケモン全て同じ日に○を記録）をした時だけ、画面下部にひっそりと一言メッセージが出て数秒で消える。「今日はいい夜だった」など。

7. **ポートレートの色がずれる**
グリッチ表現として、まれにポートレートカードが一瞬だけ RGB がずれたようなノイズエフェクトをかける。ホラーゲーム的な演出。世界観にも合う。

8. **確定セルが金色に輝く**
通常は accentColor で光るが、N=2（100%）のセルを記録した瞬間だけ一時的に金色のパーティクルが散る。通常は使われない特別なカラーで差別化。

9. **ダークライが目を光らせる**
深夜（23時〜1時）にアプリを開いた時だけ、ダークライのポートレートの目の位置に小さな赤い光のエフェクトが重なる。時刻連動の隠し演出。

10. **記録数マイルストーン**
累計50回・100回記録した瞬間に、星が一斉に中心に集まって弾けるワンショットアニメーションが走る。次のマイルストーンまでの記録数をどこかに忍ばせておく。

## このツール特有のアイデア

### 確率・データ活用系

1. **「今日逃したら次はいつ?」シミュレーター**
未記録セルを長押しすると「ここでハズレたら次回50%、ここで出現したら次回17%」という分岐が小さなポップオーバーで見える。意思決定の補助ではなく、仕組みへの理解を深める。

2. **通算出現率の表示**
全履歴から「ミュウ: 12戦8勝（66.7%）」のように実測値を表示するコーナー。理論確率との乖離が見えるとゲーム性が増す。特に長期ユーザーほど面白い数字になる。

3. **ポケモンごとの「連勝・連敗記録」**
過去最長の連続出現・連続ハズレを記録しておき、更新した瞬間にそっと表示する。「ダークライ 連続ハズレ更新中: 5回」など。

4. **月末サマリーカード**
月の3日目を記録した瞬間に「今月のダークライ: ×・○・× / 次月1日目は50%スタート」というカードが一瞬表示される。Spotifyの年間まとめのミニ版。

### ゲーム的エンゲージメント系

5. **記録ストリーク（継続記録）**
毎月イベントの記録を欠かさず入力している連続月数を表示。「12ヶ月連続記録中」。記録が抜けた月があるとリセット。継続のモチベーションになる。

6. **ハズレた時だけ出る慰めメッセージ**
×を記録した瞬間にだけ、カードの下にひっそり一言が流れる。「次は50%。夜はまだ長い。」「ダークライは気まぐれだ。」など数十パターン用意してランダム表示。

7. **他ポケモンとの出現タイミング比較**
「同じ月にミュウとダークライが同日出現した回数: 3回」のような偶然の一致を記録・表示する。意味はないが愛着が湧く。

### データ管理・共有系

8. **記録のエクスポート / インポート**
JSON または URL パラメータで全データを出力・入力できる。機種変更時のデータ移行手段として必須。localStorage だけでは機種変更で消える。

9. **スクリーンショット用モード**
「共有」ボタンを押すと余計なUIが消え、ポケモンカードと結果だけが綺麗に並んだビューに切り替わる。SNS投稿用。OGPカード的な見せ方。

10. **複数人のデータを並べて比較**
パートナーや友人の記録をローカルに持てる「プロフィール」機能。「自分」「相方」を切り替えて見る。サーバー不要でlocalStorageを名前空間で分けるだけで実現できる。
